# Soul Builder

**Gib deinem KI-Agenten eine echte Identität.** Soul Builder führt dich in 6 Fragen durch den Prozess, die Soul deines Agenten zu definieren — und generiert ein fertiges SOUL.md.

> *"Ein Agent ohne Soul ist nur ein Werkzeug."* — ag3nt.id

## App Directory Listing

### MCP Tools

| Tool | Description |
|------|-------------|
| `start_soul_builder(session_id)` | Startet Session, gibt Willkommensnachricht + erste Frage |
| `answer_question(session_id, answer)` | Nimmt Antwort, gibt nächste Frage oder Completion |
| `generate_soul(session_id)` | Generiert SOUL.md aus allen Antworten |
| `export_soul(session_id)` | Exportiert SOUL.md als plain text |

## Fragen-Flow

1. **Name:** Wie soll dein Agent heißen?
2. **Personality:** Beschreibe die Persönlichkeit in 3 Adjektiven
3. **Core Values:** Was sind 2-3 Kernwerte deines Agenten?
4. **Tone:** Kommunikationsstil (formell / locker-professionell / casual / technisch-präzise)
5. **Backstory:** Gib deinem Agenten eine Herkunft — 1-2 Sätze
6. **Signature:** Ein typischer Erkennungs-Satz (optional, "skip" möglich)
